Репозиторий по изучению RestFull архитектуры. В рамках данного учебного проекта будут изучены http протоколы и инструменты для работы с ними. 
А также разработаны соответствующие junit тесты.
Стек технологий:
<br>Spring boot REST,
<br>Spring data,
<br>MockMvc,
<br>Mockito,
<br>Junit,
<br>Liquibase.
